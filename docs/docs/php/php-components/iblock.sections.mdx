# iblock.sections

Разделы инфоблока

## Установка

Установка компонента происходит в соответствии с <a href="/php/intro#установка-компонентов">документацией</a>

## Параметры

- IBLOCK_TYPE
  - Описание: _Тип инфоблока_
  - Обязателен: **Да**
- IBLOCK_ID
  - Описание: _Идентификатор инфоблока_
  - Обязателен: **Да**
- FIRST_SORT_FIELD
  - Описание: _Поле для первой сортировки_
  - Обязателен: Нет
- FIRST_SORT_ORDER
  - Описание: _Направление первой сортировки_
  - Обязателен: Нет
- SECOND_SORT_FIELD
  - Описание: _Поле для второй сортировки_
  - Обязателен: Нет
- SECOND_SORT_ORDER
  - Описание: _Направление второй сортировки_
  - Обязателен: Нет
- SECTION_ID
  - Описание: _Идентификатор раздела элементов_
  - Обязателен: Нет
- SECTION_CODE
  - Описание: _Код раздела элементов_
  - Обязателен: Нет
- SELECTED_FIELDS
  - Описание: _Свойства/Поля элемента для выборки_
  - Обязателен: Нет
- CACHE_TIME
  - Описание: _Время кэширования_
  - Обязателен: Нет
- CACHE_GROUPS
  - Описание: _Учитывать права доступа при кэшировании_
  - Обязателен: Нет

## Результат выполнения

Массив элементов $arResult

## Содержимое _элемента_ массива $arResult

- FIELDS
  - Описание: _Поля элемента_
- PROPS
  - Описание: _Свойства элемента_
