# form.agreement

Пользовательское соглашение

## Установка

Установка компонента происходит в соответствии с <a href="/php/intro#установка-компонентов">документацией</a>

## Параметры

- COMPONENT_ID
  - Описание: _Идентификатор компонента_
  - Обязателен: **Да**
- AGREEMENT_ID
  - Описание: _Идентификатор соглашения из административного раздела_
  - Обязателен: **Да**
- FIELDS
  - Описание: _Список полей соглашения в виде строки_
  - Обязателен: Нет
- CACHE_TIME
  - Описание: _Время кэширования_
  - Обязателен: Нет
- CACHE_GROUPS
  - Описание: _Учитывать права доступа при кэшировании_
  - Обязателен: Нет

## Результат выполнения

Строка $arResult содержащая текст соглашения
