# collection

Комплексный компонент коллекции элементов

## Установка

Установка компонента происходит в соответствии с <a href="/php/intro#установка-компонентов">документацией</a>

## Параметры

- IBLOCK_TYPE
  - Описание: _Тип инфоблока_
  - Обязателен: **Да**
- IBLOCK_ID
  - Описание: _Идентификатор инфоблока_
  - Обязателен: **Да**
- SEF\_\*
  - Описание: _Параметры ЧПУ_
  - Обязателен: Опционально
- SECTION_CODE
  - Описание: _Код раздела элементов_
  - Обязателен: Нет
- SELECTED_FIELDS
  - Описание: _Свойства/Поля элемента для выборки_
  - Обязателен: Нет
- TEMPLATE_ROOT_USE_ELEMENTS
  - Описание: _Выводить на главной странице компонента список элементов (iblock.elements)_
  - Обязателен: Нет
- TEMPLATE_ROOT_USE_SECTIONS
  - Описание: _Выводить на главной странице компонента список разделов (iblock.sections)_
  - Обязателен: Нет
- CACHE_TIME
  - Описание: _Время кэширования_
  - Обязателен: Нет
- CACHE_GROUPS
  - Описание: _Учитывать права доступа при кэшировании_
  - Обязателен: Нет

## Результат выполнения

Компонент аналогичен работе комплексного компонента "Каталог"
