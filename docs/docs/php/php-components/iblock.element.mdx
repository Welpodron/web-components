# iblock.element

Элемент инфоблока

## Установка

Установка компонента происходит в соответствии с <a href="/php/intro#установка-компонентов">документацией</a>

## Параметры

- IBLOCK_TYPE
  - Описание: _Тип инфоблока_
  - Обязателен: **Да**
- IBLOCK_ID
  - Описание: _Идентификатор инфоблока_
  - Обязателен: **Да**
- ID
  - Описание: _Идентификатор элемента_
  - Обязателен: **Да если не указан `CODE`**
- CODE
  - Описание: _Код элемента_
  - Обязателен: **Да если не указан `ID`**
- SECTION_ID
  - Описание: _Идентификатор раздела элемента_
  - Обязателен: Нет
- SECTION_CODE
  - Описание: _Код раздела элемента_
  - Обязателен: Нет
- SELECTED_FIELDS
  - Описание: _Свойства/Поля элемента для выборки_
  - Обязателен: Нет
- CACHE_TIME
  - Описание: _Время кэширования_
  - Обязателен: Нет
- CACHE_GROUPS
  - Описание: _Учитывать права доступа при кэшировании_
  - Обязателен: Нет

## Результат выполнения

Массив элементов $arResult

## Содержимое _элемента_ массива $arResult

- FIELDS
  - Описание: _Поля элемента_
- PROPS
  - Описание: _Свойства элемента_
