# Начало работы с PHP библиотекой

## Установка модулей

Для установки PHP библиотеки требуется скопировать исходную папку модуля (например `welpodron.core`) в папку `modules` внутри корневой папки `local`,
после проделанной операции модуль должен появиться в списке модулей Bitrix Marketplace, затем его необходимо установить также как и устанавливаются другие модули от сторонних разработчиков.

## Установка компонентов

Для установки PHP компонента требуется скопировать исходную папку компонента (например `form.agreement`) в папку `welpodron` внутри папки `components` корневой папки `local`.
