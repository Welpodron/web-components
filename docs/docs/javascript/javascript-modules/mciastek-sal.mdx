# mciastek-sal

Анимация при скролле

:::info Информация

Данный модуль является модификацией mciastek-sal. Полная документация доступна по https://github.com/mciastek/sal

:::

## Установка / Подключение

Список PHP библиотек:

- welpodron.core

:::caution Внимание

Перед подключением модуля убедитесь, что все необходимые для требуемого модуля зависимости установлены и подключены!

:::

Установка и подключение модуля происходит в соответствии с <a href="/javascript/intro#установка-модулейкомпонентов">документацией</a>

## Использование

### Использование через HTML

```html title="html"
...
<div
  ...
  data-sal="Вид анимации"
  [data-sal-easing]
  [data-sal-duration]
  [data-sal-delay]
  ...
>
  ...
</div>
...
```

## HTML атрибуты

- Атрибут: `data-sal`
  - Описание: _Указатель вида анимации_
  - Обязателен: **Да**
- Атрибут: `data-sal-easing`
  - Описание: _Вид сглаживания анимации_
  - Обязателен: Нет
- Атрибут: `data-sal-duration`
  - Описание: _Длительность анимации_
  - Обязателен: Нет
- Атрибут: `data-sal-delay`
  - Описание: _Задержка перед анимацией_
  - Обязателен: Нет
