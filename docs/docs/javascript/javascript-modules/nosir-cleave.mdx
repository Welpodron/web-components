# nosir-cleave

Маска для полей ввода. В данной модификации используется только для телефонного номера

:::info Информация

Данный модуль является модификацией nosir-cleave. Полная документация доступна по https://github.com/nosir/cleave.js

:::

## Установка / Подключение

Список PHP библиотек:

- welpodron.form

:::caution Внимание

Перед подключением модуля убедитесь, что все необходимые для требуемого модуля зависимости установлены и подключены!

:::

Установка и подключение модуля происходит в соответствии с <a href="/javascript/intro#установка-модулейкомпонентов">документацией</a>

## Использование

### Использование через HTML

```html title="html"
...
<input ... data-tel-mask ... />
...
```

## HTML атрибуты

- Атрибут: `data-tel-mask`
  - Описание: _Указатель на использование телефонной маски_
  - Обязателен: **Да**
