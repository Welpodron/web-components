# verlok-vanilla-lazyload

Ленивая загрузка

:::info Информация

Данный модуль является модификацией verlok-vanilla-lazyload. Полная документация доступна по https://github.com/verlok/vanilla-lazyload

:::

## Установка / Подключение

Список PHP библиотек:

- welpodron.core

:::caution Внимание

Перед подключением модуля убедитесь, что все необходимые для требуемого модуля зависимости установлены и подключены!

:::

Установка и подключение модуля происходит в соответствии с <a href="/javascript/intro#установка-модулейкомпонентов">документацией</a>

## Использование

### Использование через HTML

```html title="html"
...
<img ... data-lz data-src="Путь до изображения (локальный)" />
...
```

## HTML атрибуты

- Атрибут: `data-lz`

  - Описание: _Указатель компонента_
  - Обязателен: **Да**

- Атрибут: `data-src`
  - Описание: _Путь до локально размещенного изображения_
  - Обязателен: **Да**
